# Screenshots Folder

This folder contains screenshots of the NexNote application for documentation purposes.

## Required Screenshots

Please add the following screenshots to this folder:

1. **homepage.png** - Main landing page showing feature overview
2. **chat-interface.png** - Chat interface with conversation history
3. **chat-example.png** - Example of AI response with source citations
4. **study-tools.png** - Study tools dashboard
5. **quiz-generation.png** - Quiz generation and questions
6. **quiz-results.png** - Quiz results with score and explanations
7. **flashcards.png** - Flashcard interface
8. **file-upload.png** - File upload interface
9. **calendar.png** - Calendar page (optional, if enabled)
10. **progress-dashboard.png** - Study progress analytics (optional)

## How to Take Screenshots

See the main project file `HOW_TO_ADD_SCREENSHOTS.md` for detailed instructions.

Quick steps:
1. Run the application: `.\start.ps1`
2. Press `Win + Shift + S` to capture screenshots
3. Save them to this folder with the names listed above
4. Commit and push to GitHub

## File Guidelines

- **Format**: PNG (preferred) or JPG
- **Max size**: 1MB per image
- **Resolution**: 1920x1080 or your screen resolution
- **Naming**: Use lowercase with hyphens (e.g., `quiz-results.png`)
